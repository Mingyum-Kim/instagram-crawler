# Project Brief

## 프로젝트명(가칭)
Instagram Organizer

## 한 줄 정의
인스타그램 저장 게시물/릴스/DM 정보를 자동 수집·요약·분류해서, 사용자가 놓친 정보를 다시 보게 만드는 웹 서비스.

## 해결하려는 문제
- 사용자는 콘텐츠를 저장만 하고 나중에 다시 찾지 못한다.
- 정보성 DM을 받아도 시간이 지나면 잊어버린다.
- 결과적으로 유용한 정보가 실행으로 이어지지 않는다.

## 타겟 사용자
- 일반 사용자 (개인용)

## MVP 목표
1. 저장 게시물/릴스/DM 텍스트를 자동으로 수집한다.
2. 자동 분류/요약으로 정리 부담을 없앤다.
3. 주간 웹 알림으로 재노출한다.

## 범위(In Scope)
- 웹 애플리케이션
- 온보딩 3단계 + 필수 동의
- 인스타 연동 후 초기/증분 동기화
- 자동 카테고리 분류
- 대시보드/카테고리/상세/알림 센터
- 주간 요약 생성(웹 내 알림)

## 제외(Out of Scope)
- 검색 기능
- 이메일 알림
- 팀 공유/협업 기능
- 유료 결제/구독 기능 (초기 무료 검증 단계)
